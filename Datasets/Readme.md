# Dataset

This folder contains the datasets used to build the Restaurant Analytics Dashboard.

## Files

- customers.csv – Customer information
- orders.csv – Order details
- restaurants.csv – Restaurant information
- deliveries.csv – Delivery-related data (if applicable)
- date_table.csv – Date dimension used for time intelligence (if applicable)

## Source

Dataset used for educational and portfolio purposes.

## Notes

- Data was cleaned using Power Query.
- Additional calculated columns and DAX measures were created in Power BI.
